## Daily Development Commands

```bash
# Start WordPress
ddev start

# Stop WordPress  
ddev stop

# Start Astro dev server
pnpm -C apps/astro dev

# Rebuild design tokens
pnpm so tokens

# Build for production
pnpm run build
```
