# Getting Started

Welcome to ReactorBro Stack! This guide will help you get started quickly.

## Prerequisites

- Node.js 20+
- pnpm 9+
- Git

## Installation

\`\`\`bash
# Clone the repository
git clone https://github.com/your-org/reactorbro-stack.git
cd reactorbro-stack

# Install dependencies
pnpm install

# Bootstrap the project
pnpm bootstrap
\`\`\`

## Quick Start

\`\`\`bash
# Create a new site
pnpm site:create my-site

# Activate the site
pnpm site:use my-site

# Start development server
pnpm site:dev
\`\`\`
